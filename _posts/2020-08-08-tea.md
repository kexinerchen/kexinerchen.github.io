### 按發酵程度分類：綠白黃，青紅黑
```
綠茶 ---- 不發酵，西湖龍井，黃山毛峰，黃山綠牡丹，蘇州洞庭山碧羅春，紫陽富硒毛尖，六(lù)安瓜片，信陽毛尖，太平猴魁，廬山雲霧
白茶 ---- 微發酵，福鼎白茶、壽眉、貢眉，白牡丹、白毫銀針
黃茶 ---- 輕發酵，湖南君山銀葉，黃大茶，黃小茶，黃芽茶
青茶 ---- 中發酵，即烏龍茶，四大烏龍
紅茶 ---- 全發酵，祁門紅茶，閩北正山小種(紅茶鼻祖)、金駿眉(頂級紅茶)，雲南滇紅
黑茶 ---- 後發酵，雲南普洱，廣西六堡茶，湖南黑茶，湖北黑茶，四川邊茶(雅安藏茶)

```

#### 四大烏龍
```
閩北 ---- 武夷岩茶，獨具岩韻，滋味厚重，回甘明顯。四大岩茶：大紅袍、鐵羅漢、水金龜、白雞冠。武夷肉桂，武夷水仙
閩南 ---- 鐵觀音，有天然的蘭花香，熟果味。黃金桂，閩南水仙，永春佛手
廣東 ---- 鳳凰單樅，領頭單樅
台灣 ---- 凍頂烏龍，顆粒重實。東方美人

```

#### 沖泡溫度
```
綠茶75~85度，白茶80-90度，黃茶85-95度
青茶90-100度，紅茶95-100度，黑茶100度

```

#### 一些常識
```
1> 綠茶是未經發酵，經過殺青的；白茶是輕度發酵，不殺青，經過萎凋而成
安吉白茶用了綠茶的殺青工藝，並且沒有發酵，所以從工藝上看是綠茶；安吉白茶樹種是“白葉一號”，這種樹在23度以下，長出的芽葉偏白。

2> 大紅袍是烏龍茶(青茶)，不是紅茶。大紅袍名字源於一個故事：舉子丁顯上京趕考途中腹痛，巧遇一和尚給他喝茶治病，後來考中狀元回來感謝和尚，並在茶樹上披上紅袍，大紅袍因此得名。

3> 君山銀葉是黃茶，白毫銀針是白茶。湖南洞庭湖一個叫君山的島產君山銀葉，太湖洞庭山(湖中東山西山合稱洞庭山)產洞庭碧羅春。
```
