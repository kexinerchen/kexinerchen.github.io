### 按發酵程度分類
```
綠茶 ---- 不發酵，西湖龍井，黃山毛峰，黃山綠牡丹，蘇州洞庭山碧羅春，紫陽富硒毛尖，六安瓜片，信陽毛尖
黃茶 ---- 微發酵，湖南君山銀葉，黃大茶，黃小茶，黃芽茶
白茶 ---- 微發酵，福鼎白茶壽眉、貢眉，白牡丹、白毫銀針
青茶 ---- 中度發酵，即烏龍茶，四大烏龍
紅茶 ---- 全發酵，祁門紅茶，閩北正山小種、金駿眉，雲南滇紅
黑茶 --- 後發酵，雲南普洱，廣西六堡茶，湖南黑茶，湖北黑茶，四川邊茶(雅安藏茶)

```

#### 四大烏龍
```
閩北 ---- 武夷岩茶，獨具岩韻，滋味厚重，回甘明顯。四大岩茶：大紅袍、鐵羅漢、水金龜、白雞冠。武夷肉桂，武夷水仙
閩南 ---- 鐵觀音，有天然的蘭花香，熟果味。黃金桂，閩南水仙，永春佛手
廣東 ---- 鳳凰單叢，領頭單叢
台灣 ---- 凍頂烏龍，顆粒重實。東方美人

```
